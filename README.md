# Acme Warehouse API

![CI](https://github.com/anothererikjames/acme-warehouse-api/actions/workflows/ci.yml/badge.svg)
![Coverage](https://img.shields.io/badge/coverage-94%25-brightgreen)
![Version](https://img.shields.io/badge/version-2.3.1-blue)

REST API for inventory and warehouse management across all Acme fulfillment centers.

## Overview
- Real-time stock tracking across 12 warehouse locations
- Automated low-stock alerts and replenishment triggers
- Supplier integration with EDI and REST-based vendors

## Quick Start
```bash
curl -H "Authorization: Bearer $TOKEN" https://api.acme.com/warehouse/v2/health
```
